Other Works

  • JavaScript Game Development Bootcamp Project

    Date created: February 8, 2025

    Tags: htmlcssjavascript

    This project is a 2048-inspired game developed during a Zuitt bootcamp on February 8-9, 2025 using HTML, CSS, and JavaScript. Players combine numbered tiles to achieve higher scores, with milestone prompts for 2048, 4096, and 8192. The game ends when no moves are left, displaying a game-over message. Players can restart instantly by pressing any key. Additionally, a Facebook sharing feature allows users to share their game progress.


  • Frontend Fundamentals Bootcamp: Basic Task Manager

    Date created: October 27, 2024

    Tags: htmlcssjavascript

    This web-based task manager was developed as part of OneCodeCamp Academy’s Frontend Fundamentals Bootcamp, focusing on essential frontend development skills. It enables users to add, manage, and organize daily tasks effortlessly. Completed tasks are visually distinguished with an underline and reduced opacity, making it easy to track progress. Users can also delete tasks with a smooth falling animation, symbolizing task removal. Additionally, a filtering feature allows seamless sorting of tasks by status—completed, incomplete, or all—ensuring a more organized and efficient task management experience.


  • Data Visualization Workshop

    Date created: October 5, 2024

    Tags: htmljavascriptchart.jsapi

    This project is part of a data visualization bootcamp hosted by Zuitt, focusing on real-time trend analysis using APIs. It dynamically retrieves and displays daily trending topics from X (formerly Twitter) through an API, ensuring up-to-date insights. The data is visualized using Chart.js, offering users an interactive and visually engaging representation of the most talked-about topics on social media. With automated daily updates, this project provides a seamless experience for tracking social media trends in real time.
